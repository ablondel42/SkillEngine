# Domain Components Reference

Quick reference for domain-specific components. Pick 5-10 that match your
user's core features.

---

## Dashboard / Analytics

**Core components:** StatCard, KPI, LineChart, BarChart, DataTable, FilterBar,
DateRangePicker, TrendIndicator, GaugeChart, WidgetContainer

**Typical screens:** Overview dashboard, data tables with sorting/filtering,
chart widgets, settings pages

---

## Project Management

**Core components:** KanbanBoard, TaskCard, SprintBoard, GanttChart,
TaskList, TaskDetail, AssigneePicker, PriorityBadge, DueDateIndicator,
TimeTracker, SubtaskList, CommentThread

**Typical screens:** Kanban board, task detail panel, sprint planning,
team dashboard

---

## E-commerce

**Core components:** ProductCard, ProductGallery, ProductVariants,
PriceDisplay, AddToCartButton, ShoppingCart, CartItem, CheckoutStep,
ReviewCard, RatingStars, ProductFilters, StockIndicator

**Typical screens:** Product listing, product detail, cart, checkout flow,
order history

---

## Social / Media

**Core components:** PostCard, CommentThread, LikeButton, ShareButton,
UserProfile, FeedItem, FollowButton, NotificationBadge, MessagePreview,
MediaUploader, HashtagLink, ReactionPicker

**Typical screens:** Feed, profile, post detail, notifications, messages

---

## SaaS / B2B

**Core components:** PricingCard, FeatureTable, TestimonialCard,
IntegrationCard, OnboardingStep, UsageMeter, ApiKeyDisplay, TeamInvite,
AuditLog, BillingTable, UpgradePrompt, EmptyState

**Typical screens:** Landing, pricing, onboarding wizard, dashboard,
account settings

---

## Healthcare

PrescriptionList, VitalSigns, SymptomChecker, LabResults, InsuranceCard,
CareTeam, MedicationReminder, TelehealthButton

**Typical screens:** Patient list, patient detail, appointment scheduler,
health records

---

## Finance

**Core components:** AccountSummary, TransactionRow, TransactionList,
TransferForm, BudgetCard, SpendingChart, BillReminder, InvestmentChart,

**Typical screens:** Account overview, transactions, transfers, statements,

---

## Education

**Core components:** CourseCard, LessonList, LessonViewer, QuizQuestion,
QuizResults, ProgressTracker, CertificateBadge, AssignmentCard,
GradeDisplay, DiscussionPost, VideoPlayer, CalendarView

**Typical screens:** Course list, lesson view, quiz, progress tracker,
calendar

---

## Tips for Choosing Components

1. **Start with core workflows** - What do users do most?
2. **Think about data display** - Tables, charts, cards?
3. **Consider input needs** - Forms, search, filters?
4. **Don't forget feedback** - Alerts, loaders, empty states
5. **Combine with core components** - Buttons, inputs, cards are still needed

---

## Example Combinations

**Analytics dashboard:** StatCard + LineChart + DataTable + FilterBar

**Task app:** KanbanBoard + TaskCard + AssigneePicker + DueDateIndicator

**Shop:** ProductCard + ProductGallery + ShoppingCart + CheckoutStep
